About This Project

Insurance Predictor is a modern web application designed to estimate your insurance premium based on your personal and health details. Powered by machine learning, it provides quick, reliable, and personalized premium predictions to help you make informed decisions.

  • Easy-to-use, beautiful interface
  • Instant premium predictions
  • Secure and private
  • Built with Python, Flask, and Bootstrap

Our mission is to make insurance more transparent and accessible for everyone. Whether you're planning for your future or just curious, this tool is here to help!

Project Details

Tech Stack
  • Python 3
  • Flask Web Framework
  • Bootstrap 5 for UI
  • Joblib for Model Serialization
  • HTML5, CSS3, Jinja2
Features
  • User-friendly insurance premium prediction form
  • Instant, accurate predictions using ML model
  • Prediction history tracking
  • Contact form for feedback
  • Modern, responsive design

This project demonstrates the power of data science and web development to solve real-world problems. Explore the code, try predictions, and see how machine learning can make a difference!
